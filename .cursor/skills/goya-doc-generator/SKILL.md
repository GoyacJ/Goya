# Goya 文档生成和同步

## 描述
自动生成和同步项目文档，包括 API 文档、模块文档等。

## 使用场景
- 生成 API 文档
- 生成模块 README
- 更新 CHANGELOG
- 生成架构图

## 功能清单

### 1. API 文档

基于代码生成 OpenAPI/Swagger 文档

### 2. 模块 README

为每个模块生成标准的 README.md：
- 模块介绍
- 功能特性
- 使用示例
- 配置说明

### 3. CHANGELOG

根据 Git 提交记录生成变更日志

### 4. 架构图

生成模块依赖关系图、类图等

## 使用方法

- "为 component-security 生成 README"
- "更新 API 文档"
- "生成架构图"
